# IoT-Sensors-Emulator

Эмулятор IoT-устройств 


Мой первый опыт вайб кодинга.

Источник вдохновения: 

- проект 'C2T7-MQTT SmartHome' (Python, библиотека paho-mqtt для эмуляции обмена данными между устройствами).
Отличия: тут не сценарий "умного дома", а универсальный инструмент для эмуляции разных устройств с настройкой их поведения.


- курс Интернет вещей от Академии Самсунг https://aiu.susu.ru/iot/samsung
